## 🔍 Специальный тест: Редактирование отчета

1. Перейти на вкладку **Отчеты**
2. Найти отчет и нажать кнопку **"Редактировать"** (иконка карандаша) или кнопку **"Просмотр деталей"**
3. Должна открыться форма редактирования отчета
4. **ПРОВЕРИТЬ ЧТО ВИДНЫ ВСЕ ЭТИ ПОЛЯ:**

### Обязательно видны:
- [ ] **Дата отчета**
- [ ] **Оператор** (Select dropdown)
- [ ] **Объект** (Select dropdown)
- [ ] **Установка** ⭐ КРИТИЧНО
  - Должно быть поле Select с названием "Установка"
  - Должно быть заполнено значение (например "Бауман-100")
  - Если этого поля нет - ОШИБКА ❌

### Забитые сваи (Piles):
- [ ] **Раздел видим** с заголовком "Забитые сваи"
- [ ] Видны записи типа: "Марка сваи — Количество"
- [ ] Если сваи есть - должны быть видны
- [ ] Если нет свай - должна быть кнопка "Добавить сваю"
- [ ] Если этого раздела нет - ОШИБКА ❌

### Лидерное бурение (Drilling):
- [ ] **Раздел видим** с заголовком "Лидерное бурение"
- [ ] Видны записи типа: "Тип бурения — Метры"
- [ ] Если бурения есть - должны быть видны
- [ ] Если нет - должна быть кнопка "Добавить бурение"
- [ ] Если этого раздела нет - ОШИБКА ❌

### Причины простоев (Downtimes):
- [ ] **Раздел видим** с заголовком "Простой техники"
- [ ] Видны записи типа: "Причина — Часы"
- [ ] Если простои есть - должны быть видны
- [ ] Если нет - должна быть кнопка "Добавить простой"
- [ ] Если этого раздела нет - ОШИБКА ❌
