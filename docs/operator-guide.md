# Инструкция оператора PilingTrack

Документ для оператора-сваебойщика. Описан весь рабочий процесс: от входа в приложение до отправки отчёта в Telegram.

---

## 1. Что вам нужно

- **Смартфон** (Android 9+ или iPhone 12+) **или планшет** с интернетом
- **Логин и пароль** (выдаёт администратор)
- **Объект и бригада** должны быть назначены вам администратором заранее. Без назначения вы ничего в приложении не увидите

---

## 2. Первый вход

1. Откройте в браузере: **https://[адрес-вашей-компании]**
2. На странице входа введите:
   - **Email** — выданный администратором (например, `ivanov@piling.ru`)
   - **Пароль** — выданный администратором
3. Нажмите **«Войти»**
4. **Установите приложение на главный экран** (рекомендуется):
   - **Android**: меню браузера → «Добавить на главный экран»
   - **iPhone**: значок «Поделиться» → «На экран „Домой"»

После установки приложение работает как обычное — открывается с иконки.

---

## 3. Главный экран

После входа вы видите три вкладки внизу экрана:

| Вкладка | Что показывает |
|---|---|
| **Главная** | Ваши объекты и кнопка создания отчёта |
| **Отчёт** | Форма заполнения нового отчёта |
| **История** | Все ваши отправленные отчёты с возможностью просмотра PDF |

---

## 4. Создание отчёта (главный сценарий)

### Шаг 1. Откройте форму

- Нажмите вкладку **«Отчёт»** или большую кнопку **«+»** на главной

### Шаг 2. Выберите параметры смены

| Поле | Что заполнить |
|---|---|
| **Объект** | Выберите из списка. Если списка пустого — обратитесь к администратору, вы не назначены |
| **Дата** | По умолчанию сегодня. Меняется только в редких случаях |
| **Смена** | Дневная (08:00–20:00) или Ночная (20:00–08:00) |
| **Оборудование** | Подставится автоматически из вашей бригады |
| **Помощник** | Подставится автоматически (если в бригаде назначен) |

> Если оборудование/помощник не подставились — администратор не привязал их к вашей бригаде. Свяжитесь с диспетчером.

### Шаг 3. Свайные работы

1. Нажмите **«Добавить сваи»**
2. Выберите **марку сваи** (только марки, которые есть в плане объекта)
3. Введите **количество забитых свай** за смену
4. Метраж посчитается автоматически (количество × длина сваи из марки)

> Можно добавить несколько строк, если за смену работали с разными марками.

### Шаг 4. Лидерное бурение

Если в смену было бурение под сваю:

1. Нажмите **«Добавить бурение»**
2. Выберите **тип бурения** (диаметр)
3. Введите **количество скважин** и **глубину каждой**
4. Общий метраж считается автоматически

### Шаг 5. Простои (если были)

Если оборудование стояло (поломка, ожидание, погода):

1. Нажмите **«Добавить простой»**
2. Выберите **причину** из списка
3. Укажите **длительность в часах**
4. Можно добавить **комментарий**

### Шаг 6. Сохранение и отправка

Внизу формы две кнопки:

- **«Сохранить черновик»** — отчёт остаётся у вас, можно вернуться и доработать. **Не уходит ни администратору, ни в Telegram**
- **«Отправить»** — финализирует отчёт. Меняется статус на «Отправлен», PDF автоматически уходит в Telegram-группу компании

> ⚠️ После отправки изменения вносит только администратор. Вы — нет.

---

## 5. Что происходит после отправки

В течение **1–3 секунд** после нажатия «Отправить»:

1. Отчёт сохраняется в базе и появляется у администратора
2. Автоматически генерируется PDF
3. В Telegram-группу `PilingTrack Group` приходит сообщение:
   ```
   📋 Отчёт отправлен
   📍 Объект: ВСМЖ Великий Новгород
   📅 Дата: 28.04.2026
   👷 Оператор: Иванов И.П.
   🛠 Оборудование: Liebherr LRH 100 №2
   🔩 Свай забито: 36 шт
   🌀 Бурение: 31 м.п.
   ⏸ Простои: 0 ч
   ```
   + прикреплённый PDF.

Если сообщение не пришло — обратитесь к администратору, возможно отвалился бот или нет интернета у сервера.

---

## 6. Просмотр своих отчётов

1. Вкладка **«История»**
2. Видите карточки всех ваших отчётов с краткой статистикой:
   - **🪛 36/540.0 шт/м.п.** — забито свай / общий метраж
   - **🌀 31/372.0 шт/м** — скважин бурения / метраж
3. На карточке зелёная плашка **«Отправлен»** или жёлтая **«Черновик»**
4. Нажмите **значок глаза 👁** на карточке — откроется PDF-просмотр
5. В PDF-просмотре кнопка **«Печать»** позволяет распечатать или сохранить

---

## 7. Работа без интернета (offline)

Приложение работает оффлайн:

- **Заполнить отчёт** можно без сети — он сохранится локально
- При появлении сети отчёт автоматически отправится на сервер
- В правом верхнем углу значок 🔴 (нет сети) или 🟢 (есть сеть)

> Не закрывайте приложение, пока не убедились, что отчёт отправлен (статус «Отправлен», а не «Ожидает синхронизации»).

---

## 8. Частые проблемы

| Проблема | Решение |
|---|---|
| Список объектов пустой | Администратор вас не назначил. Свяжитесь с диспетчером |
| Не вижу нужную марку сваи | Этой марки нет в плане объекта. Уточните у инженера ПТО |
| После «Отправить» отчёт не пришёл в Telegram | Подождите 5-10 секунд. Если не пришёл — проблема на сервере, сообщите администратору |
| Забыл пароль | Только администратор может сбросить — обратитесь к нему |
| Хочу исправить отправленный отчёт | Только через администратора. Объясните ему, что нужно поменять |
| Приложение тормозит | Закройте и откройте заново. Если не помогает — очистите кеш браузера |

---

## 9. Безопасность

- **Не передавайте пароль** другим людям — все ваши действия логируются на ваш аккаунт
- **Выходите** из приложения на чужих устройствах (правый верхний угол → значок выхода)
- **Не делайте скриншоты отчётов** для пересылки в личные чаты — все отчёты уже автоматически уходят в корпоративный Telegram
