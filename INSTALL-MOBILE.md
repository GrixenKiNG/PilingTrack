# 📱 PilingTrack на телефоне или планшете

PilingTrack — это веб-приложение. На смартфоне/планшете оно открывается **через
браузер** по адресу вашего сервера — без магазинов приложений и без установки
apk/ipa. Интерфейс адаптирован под мобильный экран (особенно роль «машинист»).

Для удобства можно добавить **ярлык на главный экран**, чтобы открывать
PilingTrack в один тап. Это обычный ярлык-закладка: он открывает сайт в браузере.

> Нужен интернет. Офлайн-режима сейчас нет — без сети приложение не работает.

---

## 🤖 Android (Chrome / Edge / Yandex / Samsung Internet)

1. Открой PilingTrack в браузере: `https://<адрес-вашего-сервера>`.
2. Войди в систему.
3. Нажми ⋮ (три точки) → **«Добавить на главный экран»**.
4. Подтверди название «PilingTrack».
5. Иконка появится на рабочем столе — открывай оттуда.

---

## 🍎 iOS / iPadOS (iPhone, iPad)

1. Открой PilingTrack в **Safari** (на iOS ярлык на экран умеет ставить только Safari).
2. Войди в систему.
3. Внизу нажми **«Поделиться»** (квадрат со стрелкой вверх).
4. Прокрути меню вниз → **«На экран „Домой"»** (*Add to Home Screen*).
5. Подтверди название «PilingTrack» → **Добавить**.
6. Иконка появится на главном экране — открывай оттуда.

### Особенности iOS
- При первом запуске может попросить разрешения: камера (фото к отчёту),
  геолокация. Разрешай — без них часть фич недоступна.
- Если в Safari вышел из системы и обновил страницу — открытый по ярлыку сайт
  тоже потеряет сессию (Apple шарит куки). Перезайди.

---

## ❓ Часто спрашиваемое

**«Где скачать APK / IPA?»**
Не нужно. Работа идёт через браузер — это и есть мобильная версия.

**«Что если интернета нет?»**
Сейчас приложению нужна сеть — офлайн-режима нет. Появится сеть — продолжай работу.

**«Можно поставить из Google Play / App Store?»**
Сейчас нет.

> **На будущее (пока не сделано):** PilingTrack можно превратить в полноценное
> PWA — с установкой как отдельное приложение, полноэкранным режимом, офлайн-кешем
> и push-уведомлениями. Для этого нужны web-манифест и service worker; сейчас их
> в проекте нет. Это отдельная задача — заводим, когда понадобится.
