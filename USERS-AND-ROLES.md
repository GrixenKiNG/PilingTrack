## Роли и их функции

### 🔑 ADMIN (Администратор)
**Полный доступ ко всей системе**

| Возможность | Описание |
|-------------|----------|
| `users.manage` | Управление пользователя (создание, редактирование, удаление) |
| `sites.manage` | Управление участками |
| `sites.assign_users` | Назначение пользователей на участки |
| `sites.manage_hierarchy` | Управление иерархией (поля, кусты, пикеты) |
| `reports.manage_all` | Управление всеми отчётами |
| `reports.read_all` | Чтение всех отчётов |
| `reports.export` | Экспорт отчётов (PDF) |
| `crews.manage` | Управление бригадами |
| `equipment.manage` | Управление техникой |
| `dictionary.manage` | Управление словарями (марки свай, типы бурения, причины простоев) |
| `telegram.manage` | Настройка Telegram-бота |
| `analytics.read` | Просмотр аналитики |
| `media.upload` | Загрузка медиа |
| `system.read` | Просмотр системной информации |

---

### 📡 DISPATCHER (Диспетчер)
**Управление участками и отчётами, без управления пользователями**

| Возможность | Описание |
|-------------|----------|
| `sites.manage` | Управление участками |
| `sites.assign_users` | Назначение пользователей на участки |
| `sites.manage_hierarchy` | Управление иерархией |
| `sites.read_all` | Чтение всех участков |
| `reports.manage_all` | Управление всеми отчётами |
| `reports.read_all` | Чтение всех отчётов |
| `reports.read_cross_user` | Чтение отчётов других пользователей |
| `crews.manage` | Управление бригадами |
| `crews.read` | Просмотр бригад |
| `analytics.read` | Просмотр аналитики |
| `system.read` | Просмотр системной информации |
| `media.upload` | Загрузка медиа |

**Нет доступа:**
| ❌ Нет возможности |
|-------------------|
| `users.manage` — управление пользователями |
| `equipment.manage` — управление техникой |
| `dictionary.manage` — управление словарями |
| `telegram.manage` — настройка Telegram-бота |
| `reports.export` — экспорт отчётов |

---

### 🔧 ASSISTANT (Монтёр / Помощник) — 4 чел.
**Минимальный доступ, помощь в создании отчётов**

| Возможность | Описание |
|-------------|----------|
| `reports.read` (свои) | Чтение своих отчётов |
| `reports.create` | Создание отчётов (под руководством) |

**Пользователи:** Сидоров К.А., Курнаков Н., Токсубаев Р., Солдатов А.С.

**Нет доступа:**
| ❌ Нет возможности |
|-------------------|
| Чтение чужих отчётов |
| Управление чем-либо |
| Аналитика |
| Экспорт |

## Иерархия ролей

```
ADMIN (1) — полный доступ
  └── DISPATCHER (1) — управление + аналитика, без users/equipment
        └── OPERATOR (6) — только свои отчёты
              └── ASSISTANT (4) — минимальный доступ
```
